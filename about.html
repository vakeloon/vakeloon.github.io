About us

Welcome to Simple Shop, a project created by Hniedoi Vladyslav as part of Lab 3. Our goal is to demonstrate modern web development techniques, including responsive design, smooth navigation, and clean UI.

At Simple Shop, we focus on creating a seamless shopping experience using the latest web technologies. This project is designed to showcase a structured layout, intuitive navigation, and optimized performance for both desktop and mobile users.
